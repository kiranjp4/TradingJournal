# IEC 61508 — 90-Day Alternative Track

Use this track if your focus is **industrial / general E/E/PE safety** rather than automotive.

**Structure:** IEC 61508 has 7 parts. Part 1 (general concepts) and Part 2 (lifecycle) are essential; Parts 3–4 cover HW/SW requirements.

## Phase 1 (Days 1–20): Concepts and lifecycle

| Day | Read | Exercise |
|-----|------|----------|
| 1 | 61508-1: Scope, definitions | Pick industrial item (e.g., emergency stop, burner control) |
| 2 | 61508-1: SIL, safe state, safety function | Define safety function for your item |
| 3 | 61508-1: Random vs systematic failures | List 3 examples of each for your item |
| 4 | 61508-2: Overall safety lifecycle | Draw lifecycle phases |
| 5 | 61508-2: Verification, validation, assessment | Define V-model for your item |
| 6–7 | Review | |
| 8 | 61508-2: Safety requirements specification | Write SRS with 3 requirements |
| 9 | 61508-2: Safety validation plan | Draft validation activities |
| 10 | 61508-1: SIL determination concepts | SIL target for your safety function |
| 11–20 | Parts 3–4 overview + mini FMEA | Build simplified SIL analysis |

## Phase 2 (Days 21–60): HW or SW depth

Follow same split as ISO track — HW (Part 2 tables, Part 3) or SW (Part 3 Annex A, Part 7 guidance).

## Phase 3 (Days 61–90): Integration

- Traceability: hazard → SRS → design → test
- Safety manual outline (61508-2)
- Gap analysis vs ISO 26262 if you need both

## Mapping note

| ISO 26262 | IEC 61508 |
|-----------|-----------|
| ASIL A–D | SIL 1–4 (not 1:1 mapping) |
| Safety goal | Safety function specification |
| HARA | Hazard and risk analysis (H&RA) |
| Safety case | Safety justification / assessment report |

Many engineers learn **61508 first** for fundamentals, then **26262** for automotive vocabulary and lifecycle detail.
