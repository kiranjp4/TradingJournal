How to update data anywhere

  1. Complete the Google Drive setup below (one-time, free).
  2. Click Sign in with Google.
  3. Open any sheet, click Edit Mode, make changes, then Save.
  4. Open the site on phone/laptop, sign in again, and your data will be there.

One-time Google Drive setup (free)

  1. Open Google Cloud Console.
  2. Create a project named Trading Journal KJP.
  3. Enable Google Drive API.
  4. Go to Credentials → Create Credentials → OAuth client ID.
  5. Application type: Web application.
  6. Add Authorized JavaScript origin:
    https://kiranjp4.github.io
  7. Copy the Client ID.
  8. Paste it in D:\Website\js\config.js and upload to GitHub.

Your data will be stored in Google Drive folder: TradingJournal-KJP/data/
